Collapsible

An interactive component that can be expanded or collapsed to show or hide content. Supports controlled state, disabled state, and accordion-style behavior.

Multiple Collapsibles

Multiple collapsible sections can be used independently.

Install the component using npm:

npm install @spark-ui/collapsible

Import and use the component:

import { Collapsible } from '@spark-ui/collapsible';

The collapsible accepts the following props:

  • open - Controlled open state
  • disabled - Disable interaction
  • openChange - Callback when state changes
