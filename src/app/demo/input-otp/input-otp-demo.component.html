功能特性

InputOTP 组件支持以下功能：

⌨️

自动聚焦

输入一位数字后自动聚焦到下一个输入框

📋

粘贴支持

可以直接粘贴完整的验证码

⬅️

退格导航

按退格键自动返回上一个输入框

✅

完成事件

所有位填满后触发完成事件

🔢

输入验证

数字模式自动过滤非数字字符

♿

无障碍支持

完整的 ARIA 属性和键盘导航

键盘快捷键

验证码输入支持的键盘操作：

0-9 输入数字
A-Z 输入字母（字母数字模式）
← → 导航输入框
Backspace 删除并后退
Ctrl+V 粘贴验证码

使用示例

基础用法：

<div spark-input-otp [length]="6" type="numeric"></div>

双向绑定：

<div spark-input-otp [(value)]="otp" [length]="6" type="numeric"></div>

完成事件：

<div spark-input-otp (complete)="onComplete($event)"></div>
