Multi-Session AI Chat

A multi-session AI chat interface with hybrid docked/floating session modes. SessionChatContainer sits at bottom-center, messages display in right-side dock (docked mode) or at independent positions (floating mode).

Features

  • ✅ Multiple concurrent sessions (max 5)
  • ✅ Docked mode: Shared right-side message area
  • ✅ Floating mode: Independent positioning
  • ✅ Session tabs with color customization
  • ✅ localStorage persistence
  • ✅ AI response simulation

Usage

  1. Click "Open Multi-Session Chat" to launch the page
  2. Send messages using the input at bottom-center
  3. Create new sessions (max 5)
  4. Switch between sessions using tabs
  5. Docked sessions show in right-side area
  6. Floating sessions show at independent positions (future UI)
