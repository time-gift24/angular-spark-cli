Component API Reference

Available components and their inputs.

NavMenuRootComponent

div[spark-navmenu-root]
  • disabled: boolean - Disable the menu
  • modal: boolean - Modal behavior
  • open: model<boolean> - Open state (two-way binding)

NavMenuTriggerComponent

button[spark-navmenu-trigger]
  • asChild: boolean - Use as child wrapper

NavMenuItemComponent

div[spark-navmenu-item]
  • value: string (required) - Unique identifier
  • disabled: boolean - Disable the item
  • inset: boolean - Add left inset padding
  • variant: 'default' | 'destructive'

NavMenuLinkComponent

a[spark-navmenu-link]
  • active: boolean - Active state styling

NavMenuLabelComponent

div[spark-navmenu-label]
  • inset: boolean - Add left inset padding

NavMenuContentComponent

div[spark-navmenu-content]
  • align: 'start' | 'center' | 'end'
  • side: 'top' | 'right' | 'bottom' | 'left'
  • animate: boolean - Enable animations
