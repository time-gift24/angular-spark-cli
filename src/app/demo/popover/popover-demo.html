Component API

Props

open - Controlled open state

onOpenChange - Callback when open state changes

side - Positioning side (top, right, bottom, left)

align - Alignment (start, center, end)

sideOffset - Distance from trigger in pixels

Usage

Popovers are lightweight alternatives to dialogs for displaying rich content without blocking the entire interface.

They support keyboard navigation and can be dismissed by pressing Escape or clicking outside the content area.
