键盘快捷键

单选组支持的键盘操作：

↑ ↓ 导航选项（纵向）
← → 导航选项（横向）
Space 选中当前项
Tab 移出焦点

使用示例

基础用法：

<div spark-radio-group [options]="options" orientation="vertical"></div>

双向绑定：

<div spark-radio-group [options]="options" [(value)]="selectedValue()"></div>

选项数据结构：

interface RadioOption {
  value: string;
  label: string;
  description?: string;
  disabled?: boolean;
}
