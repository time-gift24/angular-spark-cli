使用示例

基础用法：

<div spark-select [options]="options" placeholder="请选择..."></div>

双向绑定：

<div spark-select [options]="options" [(value)]="selectedValue()"></div>

选项数据结构：

interface SelectOption {
  value: string;
  label: string;
  disabled?: boolean;
}
