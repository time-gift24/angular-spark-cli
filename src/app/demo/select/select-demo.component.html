Usage Examples

Basic Usage:

<div spark-select-root [(value)]="value">
  <button spark-select-trigger placeholder="Select..."></button>
  <div spark-select-content>
    <div spark-select-item [value]="'option1'">Option 1</div>
  </div>
</div>

Option Data Structure:

interface SelectOption {
  value: string;
  label: string;
  disabled?: boolean;
}
