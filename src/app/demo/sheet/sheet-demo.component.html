This is a sheet component that slides in from the {{ side() }} side.

Sheets are dialog-like elements that slide in from the edge of the screen. They are useful for mobile navigation, forms, and other content that doesn't require a full modal.

Click the overlay or the close button to dismiss this sheet.

}

Features

  • • Four positions: Top, right, bottom, and left
  • • Overlay: Semi-transparent backdrop that closes the sheet when clicked
  • • Responsive: Adapts to different screen sizes
  • • Composable: Build sheets with header, content, and footer sections
  • • Accessible: Proper ARIA attributes and keyboard navigation

Component API

SheetContentComponent

  • • open: boolean - Controls visibility
  • • side: 'top' | 'right' | 'bottom' | 'left' - Slide-in direction

SheetOverlayComponent

  • • open: boolean - Controls visibility
  • • close: Event - Emitted when overlay is clicked
